## 📒 命令/Commands
| 命令       | 描述                                                         |
| ---------- | ------------------------------------------------------------ |
| /bwsba  | 显示插件信息|
| /bwsba help  | 显示帮助菜单|
| /bwsba reload  | 重新载入配置文件|
| /bwsba upcheck  | 检查版本更新|
| /bwsba edit <游戏>  | 编辑游戏|
| /bwsba shop list <游戏>  | 已设置商店列表|
| /bwsba shop remove <ID>  | 移除一个商店|
